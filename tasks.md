_Поточна версія: авто-перехід через 5.5с + ім'я і вік на одному екрані._
_Spec: `docs/screens.md §1` — 3 окремих кроки з кнопками._

- [ ] **Крок 1** — Ростік по центру, бабл «Ой! Привіт! Мені потрібен друг!», кнопка «Познайомитись»
- [ ] **Крок 2** — заголовок «Як тебе звати?», поле імені (автофокус), кнопка «Далі» неактивна поки порожнє
- [ ] **Крок 3** — «Скільки тобі років?», 5 кнопок-кіл (3–7), кнопка «Починаємо пригоду!» неактивна до вибору
- [ ] Прибрати `setTimeout(5500)` — перехід лише по кнопці

_Spec: `docs/screens.md §2`_

- [ ] Прогрес: «День X з 6» (не 365)

- [ ] Садок: максимум 6 листочків для MVP (не безмежно)

### LessonScreen (повна переробка)
_Поточна версія: 3 захардкоджені кроки._
_Spec: `docs/screens.md §3` — 4 фази, JSON-driven, відновлення з `currentPhase`._

**Базова структура:**
- [ ] Завантажувати контент з `src/content/days/day[N].json`
- [ ] При відкритті стартувати з `store.currentPhase` (якщо `todayDone === false`)
- [ ] Кнопка «← Назад» → HomeScreen БЕЗ скидання `currentPhase`

**Фаза 1 — Казка:**
- [ ] Планшет з текстом (`phase.text`), скролиться якщо не вміщується
- [ ] Якщо `audioUrl === null` → плашка «Аудіо буде незабаром»
- [ ] Кнопка «До вправи →» → `completePhase(1)`

**Фаза 2 — Вправа:**
- [ ] Якщо `videoUrl === null` → `textForParents.variantA` нумерованим списком
- [ ] Ритм дихання (`rhythm`): коло що розширюється/стискається (вдих → пауза → видих)
- [ ] Кнопка «Виконано ✓» → `completePhase(2)`

**Фаза 3 — Завдання:**
- [ ] `preTaskAudio`: якщо `null` → текст `preTaskAudio` на екрані 2 сек
- [ ] Лічильник «Завдання X з Y» вгорі фази
- [ ] Механіка `tap_correct` — варіанти вертикально, зелений/червоний фідбек 1.5с
- [ ] Механіка `tap_sequence` — сцена з об'єктами + лічильник + питання після всіх тапів
- [ ] Механіка `drag_match` — з'єднання ліній touch events, зелена/червона лінія
- [ ] Механіка `tap_letter` — букви на сцені, цільова мигає
- [ ] Механіка `trace_letter` — Canvas/SVG, перевірка ≥ 80% контуру
- [ ] Бонусні завдання (`bonus`) — показувати після основного

**Фаза 4 — Фінал:**
- [ ] Ростік по центру, радісний, підстрибує
- [ ] Анімація: листочок вилітає → приземляється в садок внизу
- [ ] Прогрес «День X з 6» (вже оновлений через `completeToday`)
- [ ] Кнопка «Завдання виконано! 🌱» → `completeToday()` → home
